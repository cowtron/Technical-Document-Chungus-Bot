Introduction

Chungus bot is a discord bot written using the discord.js documentation set.
The goal of the bot is to be a playground for the given developer, and to be utilized to have a fun and playful environment with friends in a private discord.

What you need to know

This guide assumes you have the following:

  • An IDE or proper text editor for Javascript
  • Basic Javascript understanding
  • And installation of the latest version of Node.js
  • An understanding of Linux Terminal OR Windows Command Prompt

Bot Token

Before starting the bot be sure to edit to decied if you wish to run this program on your own bot, or add the CHUNGUS BOT to your discord server.

  1. If you decided to use your own bot be sure to acess the auth.json file in the directory

  2. Open the file, and find the following code auth="askdjfio312lkf1"

  3. Edit the given string portion to your own discord bot authentifcation token

How to run

Access your terminal or command prompt and find access in the directory.
You can use the command cd to gain acess into that directory.
For example:

cd/Documents/Github/ChungusBot

Running the following command in Windows command prompt will jump through the documents folder, the github folder, and land inside the view of the Chungus Bot folder.


Inside the folder run the following command

node bot.js

After executing, the CHUNGUS BOT will now be connected to the given authentifcation token.

How to use

Chungus functions can be called using the ! commands in a discord text channel where Chungus Bot has access into. For example:


!ping

When input in the text channel, will have the Bot reply to the user Continue down below to functions to see a list of possible commands from the chungus bot.
